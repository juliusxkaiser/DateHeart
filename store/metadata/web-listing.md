# DateHeart Web Listing

## Name
DateHeart

## Short Description
One tap for a concrete date idea.

## Description
DateHeart gives couples practical, specific date ideas in seconds. Tap the animated heart, get a plan with budget, duration, preparation notes and tags, then save the ideas you actually want to do.

The app includes a large idea library, filters for category, budget and duration, favorites, history, 22 verified language packs and local pricing for the optional one-time no-ads purchase.

## Key Points
- Concrete date ideas, not vague suggestions
- 35,002 ideas per verified language
- Favorites and history
- Budget, duration and category filters
- Localized prices and payment restore
- PWA-ready with installable icons and service worker

## Target Audience
Couples looking for low-friction, practical date planning.
